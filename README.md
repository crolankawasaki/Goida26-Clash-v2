# 🚀 Goida 26 by Crolan

Готовая подписка для VPN‑приложений на ядре **Clash Meta**.  
Внутри — все серверы из актуального списка Goida 26 в одной простой группе.

---

## 🔗 Ссылка для подключения
https://raw.githubusercontent.com/crolankawasaki/Goida26-Clash-v2/main/config.yaml

Добавь её в своё приложение — и пользуйся.



---

## 📱 Приложения, которые подойдут

Подписка работает только с клиентами, поддерживающими **Clash Meta** (VLESS, Reality, Hysteria и др.).

### Android
- [**Karing** (APK с GitHub)](https://github.com/KaringX/karing/releases/)

### iOS
- [**Karing** (App Store)](https://apps.apple.com/app/karing/id6470924433)
- [**Stash** (App Store)](https://apps.apple.com/app/stash/id1596063349)

### Windows / macOS
- [**Clash Verge**](https://github.com/clash-verge-rev/clash-verge-rev/releases)
- [**Mihomo Party**](https://github.com/mihomo-party-org/mihomo-party/releases)

> ❗ FLClashX, Clash Mi и старые Clash **не подходят** — они не умеют работать с современными протоколами.

---

## 🛠 Как подключить (2 шага)

1. Установи любое приложение из списка выше.
2. В разделе «Подписка» или «Профиль» вставь ссылку:  
   `https://raw.githubusercontent.com/crolankawasaki/Goida26-Clash-v2/main/config.yaml`

После синхронизации появится группа **«Goida 26 »** со всеми серверами. Выбирай любой — и используй.

---

## 🔄 Как обновляются серверы

- Специальный скрипт (GitHub Actions) каждый час забирает свежий список из источника.
- Тебе **ничего не нужно делать** — просто обнови подписку в приложении, когда захочешь.

---

## 🆘 Если не работает

1. Проверь, что используешь **нужное приложение** (см. список выше).
2. Удали старую подписку и добавь заново — кэш может мешать.
3. Если ничего не помогло — напиши в [Issues](https://github.com/crolankawasaki/Goida26-Clash-v2/issues).
